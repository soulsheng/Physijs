Physics plugin for three.js 插件描述

Physijs brings a very easy to use interface to the three.js framework. One of the reasons three.js is so popular is because it is so incredibly easy for graphics newbies to get into 3D programming. Physijs takes that philosophy to heart and makes physics simulations just as easy to run. In fact, there are just five easy steps that must be taken to make a 3D scene come alive.

three.js最大的特点就是简单易用，物理动力学插件Physics继承这个优点，只需要五步就能让你的三维场景活起来。

How does Physijs work? 物理插件功能原理

Physijs is built on top of ammo.js (although there is also a cannon.js branch) and runs the physics simulation in a separate thread (via web worker) to avoid impacting in your application's performance and taking up your your 3D rendering time.

插件在一个新的线程里面运行，不会占用渲染时间，所以避免影响原来的渲染效率。

Who is this for? 这个插件能用来做什么

但愿你熟悉three.js，并且想加入物理引擎功能，这个插件非常适合你. 不用改变原有设计-简单地加入一些额外的碰撞体就能获取一个动态的模拟环境。

Features 特性

  • Support for multiple object shapes, including convex objects and heightfields. 支持模型和地形
  • Material system provides simple control over friction and restitution ("bounciness") 支持摩擦和弹性
  • Integrated collision detection and events 集成碰撞检测
  • Compound objects using the hierarchy system in three.js 模型分组嵌套
  • Full constraint system including Point, Hinge, Degree of Freedom, and more 支持主从约束
  • Vehicle systems 车辆模拟
  • Rotations using either euler or quaternion systems - your preference 旋转支持欧拉角和四元数两种模式
  • Built seamlessly on top of three.js to keep the same convention and coding style 无缝集成three.js

支持物理插件的网页浏览器，IE不支持
